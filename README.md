# 🛠️ Sistema de Vistoria de Catraca

Sistema desenvolvido para controle e vistoria das catracas dos passageiros em linhas operacionais, permitindo o registro diário e análise do uso.

---

## 📋 Sobre o projeto

Este sistema tem como objetivo facilitar o acompanhamento e a validação de:

- ✅ Catracas físicas  
- ✅ Catracas eletrônicas  
- ✅ Registro do uso diário pelos passageiros  

A proposta é centralizar informações e melhorar a confiabilidade dos dados operacionais.

---

## 🎯 Objetivo

Registrar e monitorar o uso das catracas pelos passageiros, garantindo dados confiáveis para análise operacional e tomada de decisão.

---

## 🚀 Funcionalidades

- 🔍 Registro de vistoria de catracas  
- ⚙️ Identificação de tipo de catraca (física ou eletrônica)  
- 📊 Organização de dados para análise futura  
- 🧾 Histórico de verificações  

---

## 🧑‍💼 Público-alvo

Este sistema é voltado para:

- Empresas de transporte público  
- Equipes operacionais  
- Supervisores e gestores das linhas
